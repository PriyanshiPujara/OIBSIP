# Task 10: Network Security Assessment

## Objective

To perform a network security assessment using Nmap and Wireshark, identify potential security risks, and provide recommendations for improving network security.

## Tools Used

- Nmap
- Wireshark
- Windows 11

## Files Included

### 1. network_security_assessment.md
Detailed security assessment report containing findings, analysis, and recommendations.

### 2. nmap_results.txt
Results of the Nmap network scan showing open ports and services.

### 3. wireshark_capture.pcap
Network traffic capture file generated using Wireshark.

## Activities Performed

- Scanned the local system using Nmap.
- Identified active ports and services.
- Captured network traffic using Wireshark.
- Analyzed captured packets.
- Evaluated potential security risks.
- Prepared a security assessment report.

## Conclusion

The assessment demonstrated how network scanning and packet analysis tools can be used to identify security weaknesses and improve the overall security posture of a network.
