# Task 8: Capture Network Traffic with Wireshark

## Objective

To capture and analyze network traffic using Wireshark.

## Tool Used

- Wireshark
- Windows 11

## Procedure

1. Installed Wireshark and Npcap.
2. Selected the active network interface.
3. Started packet capture.
4. Generated web traffic by visiting HTTP websites.
5. Applied the HTTP filter.
6. Analyzed captured packets.
7. Saved the capture file as wireshark_capture.pcap.

## Packet Analysis

The captured HTTP packets showed:

- HTTP GET requests
- Host information
- User-Agent details
- Source and destination IP addresses

Wireshark successfully captured and displayed network traffic for analysis.

## Conclusion

Wireshark is a powerful packet analysis tool used for monitoring, troubleshooting, and analyzing network communications.
